# nwuspider

## A small spider for northwest university china

Build on Scrapy

Information Source:

* [nwu.edu.cn](http://www.nwu.edu.cn)
* [ipc.nwu.edu.cn](http://ipc.nwu.edu.cn)

## Requirement

Python 3.5 or above
Scrapy 1.2.1 or above

## How to run

* ```scrapy crawl [spider_name]```
* ```python main.py```
